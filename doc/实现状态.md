## 一、CSS 选择器

### ✅ 已实现

| 选择器类型 | 示例 | 说明 |
|:--|:--|:--|
| 类型选择器 | `div`, `p`, `h1` | 匹配标签名 |
| 通用选择器 | `*` | 匹配所有元素 |
| 类选择器 | `.box`, `.highlight` | 支持多类名（`.foo.bar`） |
| ID 选择器 | `#header` | |
| 属性选择器 | `[id]`, `[type="text"]`, `[attr^="val"]`, `[attr$="val"]`, `[attr*="val"]`, `[attr~="val"]`, `[attr|="val"]` | 存在/精确/子串/列表/连字符匹配 || 伪类（部分） | `:first-child`, `:last-child`, `:nth-child(n)`, `:nth-last-child`, `:first-of-type`, `:last-of-type`, `:only-of-type`, `:only-child`, `:empty`, `:link`, `:any-link`, `:hover`, `:focus`, `:active` | 结构性伪类 + link + 交互状态 || 后代组合器 | `div span` | 祖先-后代 |
| 子组合器 | `div > p` | 直接父子 |
| 相邻兄弟 | `h1 + p` | 直接相邻 |
| 后续兄弟 | `h1 ~ p` | 任意后续兄弟 |

### ❌ 未实现

| 选择器 | 说明 |
|:--|:--|
| `:not()` | 否定伪类（katana 已解析但未匹配） |
| `::before`, `::after` | 伪元素（katana 已解析但未生成内容） |

## 二、CSS 属性

### ✅ 已实现并有视觉效果

| 属性 | 支持值 | 渲染效果 |
|:--|:--|:--|
| `display` | `block`, `inline`, `flex`, `grid`, `table`, `none` | grid 降级为 block |
| `width` / `height` | `px`, `%` | 字符单元格 |
| `padding` | 简写 `N` / `N M` / `N M O P` | 填充背景色 |
| `margin` | 同上 | 透明间距 |
| `border` | 简写 + `px` | Unicode 框线字符绘制 |
| `border-style` | `solid`, `dashed`, `dotted`, `double`, `heavy`, `rounded`, `groove`/`ridge`/`inset`/`outset` | 分别用不同框线字符绘制；groove 等降级为 solid |
| `border-color` | `#RRGGBB`, 命名色 | |
| `border-width` | 数值 | |
| `color` | `#RRGGBB`, `#RGB`, 16 种命名色 | 前景色 |
| `background-color` | 同上 | 背景色填充 |
| `text-align` | `left`, `center`, `right` | 文本对齐 |
| `font-weight` | `bold`, `700`, `bolder` | 终端粗体 ANSI |
| `text-decoration` | `underline` | 终端下划线 ANSI |
| `flex-direction` | `row`, `column` | |
| `justify-content` | `start`, `center`, `end`, `space-between`, `space-around`, `space-evenly` | |
| `align-items` | `start`, `center`, `end`, `stretch` | |
| `gap` | 数值 | flex 子元素间距 |
| `flex-grow` | 数值 | 剩余空间分配 |
| `flex-shrink` | 数值 | 解析但未使用 |
| `flex-basis` | `px` | |
| `overflow` | `hidden` | 递归裁剪子节点内容 |

### ⚠️ 解析继承但无视觉效果

| 属性 | 说明 |
|:--|:--|
| `font-family` | 终端无法改变字体 |
| `font-size` | 终端字体固定 |
| `font-style` | 终端无斜体 |
| `line-height` | 固定 1 行 |
| `text-indent` | 未渲染 |
| `visibility` | 未处理 |
| `word-spacing` | 未渲染 |
| `letter-spacing` | 未渲染 |
| `white-space` | 未处理 |
| `list-style` | 无列表标记 |
| `cursor` | TUI 无关 |

### ❌ 未实现

| 属性 | 原因 |
|:--|:--|
| `float`, `clear` | Flexbox 可替代 |
| `position: absolute/fixed` | 需层叠上下文 |
| `z-index` | 子元素天然覆盖父元素 |
| `transform`, `transition`, `animation` | 终端无意义 |
| `vertical-align` | align-items 已足够 |
| `opacity` | 终端不支持透明度 |
| `box-shadow` | 终端不支持 |
| `border-radius` | 终端无法圆角 |
| `grid-template-*` | grid 降级为 block |

## 四、渲染能力

### ✅ 已实现

| 功能 | 说明 |
|:--|:--|
| 24-bit 真彩色 | ANSI `\033[38;2;R;G;Bm` / `\033[48;2;R;G;Bm` |
| 粗体 | ANSI `\033[1m` |
| 下划线 | ANSI `\033[4m` |
| 边框绘制 | Unicode 框线字符，6 种样式不同字符：solid=`─│┌┐└┘`、dashed=`╌╎`、dotted=`┄┆`、double=`═║╔╗╚╝`、heavy=`━┃┏┓┗┛`、rounded=`─│╭╮╰╯` |
| 表格边框交点 | 自动识别 `┼├┤┴┬` 等交/分叉字符，替代独立单元格边框覆盖 |
| 文本居中/右对齐 | 按内容宽度计算偏移 |
| 换行 (`<br>`) | `\n` → 新行 |
| 滚动 | `screen_scroll_to/by/clamp` |
| 显示隐藏 | `display: none` |
| 内容裁剪 | `overflow: hidden` 递归裁剪子节点 |
| 文本截断 | 单行超出宽度显示 `…` |
| 焦点指示器 | 选中元素外围绘制条状高亮 |
| 状态栏 | 底部固定行，显示交互反馈信息 |
| 输入框编辑 | 键盘输入字符到 `g_input_buf`，实时更新 |
| **UTF-8 多字节字符** | `Cell.ch` 改为 `uint32_t`，支持中文/Emoji |

### ❌ 未实现

| 功能 | 说明 |
|:--|:--|
| 重叠元素层叠 | 依赖 DOM 顺序 |
| `display: inline-block` | 未实现该中间模式 |
| 交互重排时内存泄漏 | 伪类触发 layout rebuild，旧 tree 未 free |
