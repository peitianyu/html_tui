## 五、架构概览

```
HTML ──→ Gumbo 解析 ──→ DOM 树
CSS  ──→ Katana 解析 ──→ 样式表
              │
     ┌────────┴────────┐
     │  styletree.h    │  选择器匹配 + 优先级 + 继承 + 内联样式
     │  StyledNode     │  每个 DOM 节点的 computed styles
     └────────┬────────┘
              │
     ┌────────┴────────┐
     │  layout.h       │  display 解析 + 盒模型 + Flexbox + 文本提取
     │  LayoutNode     │  每个节点的 x/y/w/h/padding/margin/border
     └────────┬────────┘
              │
     ┌────────┴────────┐
     │  render.h       │  Cell 缓冲区 + 递归绘制 + ANSI 输出
     │  Screen         │  80x24 字符网格 → 终端
     └─────────────────┘
```
